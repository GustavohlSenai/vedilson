HTTP (Hypertext Transfer Protocol) - IoT

O que é?

Protocolo padrão da web, adaptado para transferir dados entre dispositivos IoT e a nuvem.

Como funciona?

Modelo Cliente-Servidor clássico (Requisito/Resposta). O dispositivo envia um POST/GET para um servidor web.

Características Técnicas:

Alcance: Depende da infraestrutura IP (Wi-Fi, Celular).
Energia: Alto consumo (não ideal para baterias pequenas).
Taxa de Dados: Alta.

Vantagens:

Fácil integração com aplicações web, seguro (HTTPS).

Desvantagens:

"Pesado" (cabeçalhos longos), alto consumo de energia.

Aplicações Reais:

Câmeras IP, Atualização de Firmware, APIs de automação residencial.
